# ALBOOM

A web application for users to review albums.

## How to Run

To populate the sample data, use the following command:
```bash
cd server/db
node populatedb.js
```

To download dependencies, use the following commands:
```bash
cd client
npm install
```
```bash
cd server
npm install
```

To run, use the following commands:
```bash
cd client
npm run dev
```
